# 💰 Employee Salary Prediction

A Machine Learning project that predicts an employee's salary category based on demographic and work-related features. The project covers the complete ML pipeline, from data preprocessing to deployment with Streamlit.

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training & Evaluation
* Interactive Streamlit Web App

## 🛠️ Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib
* Matplotlib

## 📂 Project Structure

```text
├── app.py
├── salary_model.pkl
├── scaler.pkl
├── label_encoders.pkl
├── requirements.txt
└── README.md
```

## ▶️ Installation

```bash
git clone https://github.com/your-username/Employee_Salary_Prediction.git
cd Employee_Salary_Prediction
pip install -r requirements.txt
```

## ▶️ Run the App

```bash
streamlit run app.py
```

## 📈 Model Workflow

1. Load and preprocess the dataset.
2. Encode categorical features.
3. Scale numerical features.
4. Train the machine learning model.
5. Evaluate model performance.
6. Deploy with Streamlit.
